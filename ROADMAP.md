# Pinterest go-to-market

## Use case
Вынести “Заставьте текст продавать” из статьи в рабочие правила для всех выходов на Pinterest:
- **Pins**: заголовок/описание по Кейплзу, не красивые слова.
- **Boards**: названия по выгоде/интересу, не абстрактно.
- **Articles-ссылки**: анкор/U-блоки тоже по личной выгоде.
- **Профиль**: био по результату, не по “личному блогу”.

## Constraints
- Бюджет: 0 для старта.
- Пилот/Double opt-in + compliance: не массировать, не давать “гарантии дохода”.
- Запрещены слова-оболочки: лучший, быстрый, удобный, качественный, эффективный, уникальный.
- Допустимы только факты/цифры/сроки или явное обещание пользы (“узнаете”, “сравниваете”, “распечатаете”).

## Implementation
- Brand kit: цвет, шрифт, watermark, шаблоны под вертикали.
- Templates: карточка, карусель, видео-пин.
- UTM + Pinterest tag.
- Queue + dedupe + alt/description.
- Пилот ≤10–15 пинов, 1 аккаунт, 1 неделя.
