# Pinterest go-to-market

## Use case
Вынести “Заставьте текст продавать” из статьи в рабочие правила для всех выходов на Pinterest:
- **Pins**: заголовок/описание по Кейплзу, не красивые слова.
- **Boards**: названия по выгоде/интересу, не абстрактно.
- **Articles-ссылки**: анкор/U-блоки тоже по личной выгоде.
- **Профиль**: био по результату, не по “личному блогу”.

## ДОФАМИНОВЫЕ КАРТИНКИ
Правила визуала для Pinterest и постов:
- Обычная жизнь + один физически невозможный слой: квартира, стол, лифт, магазин.
- Один сильный прием: масштаб, тело, предмет с нарушенной физикой, странное поведение.
- Натуральность важнее вылизанного AI-глянца.
- Для экспертов: избегать стоковых образов и клише 2014.
- Если используете лицо эксперта: допускается микс через ImageGen/Pollinations+Pillow.

## ДОФАМИНОВЫЕ ПОСТЫ
Тактика форматирования контента по формуле: быстрый крючок + маленькая награда + ощущение движения.
- Формат для хаотичных экспертов: 3 предложения, нет воды, действие ≤30 секунд.
- Запрещена вода: нет “очевидно”, нет общих рассуждений.
- Борьба с “проклятием знания”: не пишите как методичка для коллеги; пишите как другу, который устал после работы.
- Пример плохо: “проверьте оффер, сепарацию, белки и жиры”.
- Пример хорошо: “почему в 22:30 хочется именно бутерброд, хотя днём вы ели салат?”.
- Реализуется для: Pinterest, треды, email subject, заголовки статей.
- Если есть черновик эксперта: можно переписывать в этот формат через GPT, потом 3 минуты ручной правки.

## ВИДЕО-КОНТЕНТ / SHORTS / REELS
- Есть готовый скилл сценариста для коротких видео (рилсы/шортсы/YouTube Shorts).
- Ссылка на скилл: https://drive.google.com/drive/folders/1MgvrBWQAQllDuTxaVSH7TwJE6WvCocbn
- Установка: скачать файлы в папку с именем навыка, подключить в Codex/OpenAI.
- Активация: по ключевым словам или явный запрос “примени скилл сценариста и напиши сценарий для...”
- Особенность: всегда требует 1–3 итерации “прожарь” для финального сценария.
- Использование для Kredato: сценарии по финансам/страхованию/заработку, 30–60 секунд, дофаминовый формат.

## ДОФАМИНОВЫЙ ЗАКРЕП
Принцип: закреп — это дверь в канал, не мемориальная доска.
Ответ на 4 вопроса за 10 секунд:
1. я вообще туда пришёл?
2. что здесь можно решить?
3. с чего мне начать именно с моей ситуации?
4. куда идти, если хочу решить это с вами?

Формат:
- 1 пост про вход.
- 3–4 поста под разные ситуации.
- 1 пост про продукт.
- 1 кейс/доказательство.
- 1 понятный CTA.
Правило: не биография эксперта, не 89 ссылок, не свалка. Только маршрут по ситуации.

## Constraints
- Бюджет: 0 для старта.
- Пилот/Double opt-in + compliance: не массировать, не давать “гарантии дохода”.
- Запрещены слова-оболочки: лучший, быстрый, удобный, качественный, эффективный, уникальный.
- Допустимы только факты/цифры/сроки или явное обещание пользы (“узнаете”, “сравниваете”, “распечатаете”).

## Implementation
- Brand kit: цвет, шрифт, watermark, шаблоны под вертикали.
- Templates: карточка, карусель, видео-пин.
- UTM + Pinterest tag.
- Queue + dedupe + alt/description.
- Пилот ≤10–15 пинов, 1 аккаунт, 1 неделя.
