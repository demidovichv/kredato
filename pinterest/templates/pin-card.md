# ШАБЛОН ДОФАМИНОВОГО ПИНА

## Заголовок
- Формула: {{выгода}} + {{срок/цифра}} + {{конкретика}}
- Без воды: нет «очевидно», нет общих рассуждений.
- Без абстракций: нет «лучший», «эффективный», «качественный».

## Текст на картинке
- Одна мысль: максимум 6–8 слов.
- Конкретика: цифра, срок, чек-лист, таблица.
- Формат: обычная жизнь + 1 сбой / натуральность.

## Описание
- Крючок: проблема/цифра/контраст.
- Польза: 1–2 предложения по Кейплзу.
- CTA: одно лёгкое действие.
- Хэштеги: максимум 5 релевантных.

## Alt
- Один конкретный факт: что изображено и зачем.
- Без «картинка про финансы».

## Ссылка
- Внутренняя: /fin/vkldy/... /earning/...
- С UTM: ?utm_source=pinterest&utm_medium=social&utm_campaign=...
- Нет прямых партнёрских ссылок в публичном пине.

## Проверка
- [ ] Нет слов-оболочек
- [ ] Есть цифра/срок/пример
- [ ] Есть действие ≤30 секунд
- [ ] Едит на рубрику/статью, а не на главную
